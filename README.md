## Video Classroom Assessment

Локальный инструмент на Python для анализа видеозаписей уроков и помощи учителю в оценке учеников по пяти критериям:

- Лидерство
- Коммуникация
- Саморефлексия
- Критическое мышление
- Самоконтроль

### Быстрый старт

1. Установите Python 3.10+ и FFmpeg (должен быть доступен в `PATH`).
2. Создайте и активируйте виртуальное окружение.
3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Запустите интерфейс:

```bash
streamlit run ui/app.py
```

### Структура проекта (MVP)

- `rubric/` — описание рубрики и базовые структуры данных.
- `video_processing/` — извлечение кадров, детекция и трекинг учеников.
- `audio_processing/` — извлечение звука, распознавание речи и диаризация (в первой версии частично заглушки).
- `features/` — агрегация признаков на уровне ученика.
- `scoring/` — правила перевода признаков в уровни по рубрике.
- `ui/` — простой интерфейс на Streamlit.
- `data/` — локальное хранилище видео и промежуточных артефактов (не коммитится в VCS).

### Конфиденциальность

Все вычисления выполняются локально. Видео и извлечённые признаки не отправляются во внешние сервисы. Ответственность за юридические аспекты (согласие на съёмку и обработку) лежит на пользователе.
